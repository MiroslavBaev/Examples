

ODV3 - Change Log


Release Notes - Object Dictionary - Version V3.3.5.0
** Bug
    * [PSOBJD-67] - ODV3 Write object request fails when total data bytes are not specified
    * [PSOBJD-72] - Unexpected write access to address 0x00000010 after ODV3_WRITE_OBJECT_RES

** Change
    * [PSOBJD-68] - Defines for object codes are missing
    * [PSOBJD-70] - Remove determination of packet size over queue handle


Release Notes - Object Dictionary - Version V3.3.4.0
** Bug
    * [PSOBJD-71] - ODv3 Timeout reached even if the real timeout was not elapsed


Release Notes - Object Dictionary - Version V3.3.3.0
** Bug
    * [PSOBJD-69] - when more than 32 indications are active in parallel it crashes the running firmware

** Change
    * [PSOBJD-66] - update to rcX v2.1.10.0 and v2.0.8.24


ODV3 - V3.3.2.0
===================================
- 0022944: [Software] adjusted to new waf definition
- 0022317: [Software] abort request send by ODv3 despite not being needed
- 0021939: [Software] usMaxFieldLen has to be supported when object have exactly 0 byte of data (e.g. OCTET_STRING)
